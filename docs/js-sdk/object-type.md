# 对象类型

```typescript
export enum ObjectType {
  UNKNOWN = 'unknown',
  FOLDER = 'folder',
  IMAGE = 'image',
  VIDEO = 'video',
  AUDIO = 'audio',
  DOCUMENT = 'document',
}
```

## UNKNOWN

未知类型, `ObjectType.UNKNOWN`

## FOLDER

文件夹类型, `ObjectType.FOLDER`

## IMAGE

图片类型, `ObjectType.IMAGE`

## VIDEO

视频类型, `ObjectType.VIDEO`

## AUDIO

音频类型, `ObjectType.AUDIO`

## DOCUMENT

文档类型, `ObjectType.DOCUMENT`
